# LA-proyecto-final
Ejemplo de proyecto final para el modulo Logica &amp; Algoritmos, en el que se plicaran los conceptos y temas vistos en clase

## Descripcion
Este proyecto es una aplicacion basica desarrollada en JS que permite gestionar y analizar el inventario de platillos de un restaurante

Su objetivo es servir como prectiva para aprender conceptos y aplicar logica y algoritmos
